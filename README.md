# PPC Dashboard

A Vite, React, and TypeScript starter for tracking paid marketing performance.

## Scripts

- `npm install` installs dependencies.
- `npm run dev` starts the local development server.
- `npm run build` type-checks and builds the app.
- `npm run lint` runs ESLint.

## Project Structure

- `src/App.tsx` contains the dashboard shell.
- `src/components` contains reusable dashboard UI.
- `src/data` contains starter campaign data.
- `src/lib` contains utility helpers.
